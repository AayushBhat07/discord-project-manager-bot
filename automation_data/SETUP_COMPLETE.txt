═══════════════════════════════════════════════════════════════════════════════
⚙️ HOW IT WORKS
═══════════════════════════════════════════════════════════════════════════════

1. Bot checks: "Did I commit today?"
   - If yes → Exit (nothing to do)
   - If no → Continue to step 2

2. Load development plan
   - Get current day (1-15)
   - Find pending tasks for today

3. Implement next task
   - Create/modify code files
   - Add working functionality
   - Use proper code templates

4. Commit changes
   - Stage files: git add .
   - Commit: git commit -m "feat(...): description"
   - Push: git push origin features-dev

5. Update plan
   - Mark task as completed
   - Save progress

6. Wait & repeat
   - Random delay (30-90 minutes)
   - Continue until 3 commits made today
